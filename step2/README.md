# 나라와 수도 맞추기 퀴즈 게임

## 1. 프로젝트 개요

이 프로젝트는 터미널에서 실행되는 Python 콘솔 퀴즈 게임입니다.  
사용자는 메뉴를 통해 퀴즈를 풀고, 새로운 퀴즈를 추가하고, 등록된 퀴즈 목록을 확인하고, 최고 점수 및 플레이 기록을 확인할 수 있습니다.

이 프로그램은 Python의 기본 문법, 클래스 구조, 파일 입출력, JSON 데이터 저장, 예외 처리, Git 브랜치 작업을 직접 연습하기 위해 제작되었습니다.

---

## 2. 퀴즈 주제 선정 이유

퀴즈 주제는 **나라와 수도 맞추기**로 정했습니다.

이 주제를 선택한 이유는 다음과 같습니다.

- 문제를 만들기 쉽고 직관적이다.
- 객관식 4지선다형으로 구성하기 좋다.
- 너무 어렵지 않아 누구나 테스트하기 쉽다.
- 퀴즈 추가 기능을 구현할 때도 새로운 문제를 확장하기 편하다.

기본 퀴즈는 대한민국, 일본, 중국, 미국, 프랑스, 영국의 수도 문제로 구성되어 있습니다.

## 4. 주요 기능

### 4-1. 메뉴 기능
프로그램 실행 시 아래 메뉴가 출력됩니다.

1. 퀴즈 풀기
2. 퀴즈 추가
3. 퀴즈 목록
4. 점수 확인
5. 종료

사용자는 번호를 입력하여 원하는 기능을 선택할 수 있습니다.

### 4-2. 퀴즈 풀기
- 저장된 퀴즈를 순서대로 출제합니다.
- 각 문제는 4개의 선택지로 구성됩니다.
- 사용자가 정답 번호를 입력하면 정답/오답 여부를 알려줍니다.
- 모든 문제를 푼 뒤 총 점수를 출력합니다.
- 최고 점수를 자동으로 갱신합니다.
- 플레이 기록도 함께 저장됩니다.

### 4-3. 퀴즈 추가
- 새로운 문제를 직접 등록할 수 있습니다.
- 문제, 선택지 4개, 정답 번호, 힌트를 입력받습니다.
- 추가된 퀴즈는 즉시 `state.json`에 저장됩니다.

### 4-4. 퀴즈 목록
- 현재 저장된 퀴즈 문제 목록을 확인할 수 있습니다.
- 등록된 퀴즈가 몇 개인지도 함께 확인할 수 있습니다.

### 4-5. 점수 확인
- 최고 점수를 확인할 수 있습니다.
- 지금까지의 플레이 기록도 확인할 수 있습니다.

### 4-6. 안전한 종료 및 예외 처리
다음과 같은 예외 상황을 처리하도록 구현했습니다.

- 빈 입력
- 숫자가 아닌 입력
- 허용 범위를 벗어난 숫자 입력
- `Ctrl + C` 입력
- `EOFError`
- `state.json` 파일 없음
- `state.json` 파일 손상
- 파일 읽기/쓰기 오류

이러한 경우 프로그램이 바로 비정상 종료되지 않도록 하고, 가능한 경우 데이터를 저장한 뒤 안전하게 종료되도록 구성했습니다.

## 5. 클래스 구조

이 프로젝트는 3개의 클래스로 구성되어 있습니다.

### 5-1. `Quiz`
개별 퀴즈 1개를 표현하는 클래스입니다.

#### 주요 속성
- `question`: 문제
- `choices`: 선택지 4개
- `answer`: 정답 번호
- `hint`: 힌트

#### 주요 메서드
- `display()` : 문제와 선택지 출력
- `is_correct(user_answer)` : 정답 여부 확인
- `to_dict()` : JSON 저장용 딕셔너리 변환
- `from_dict()` : 딕셔너리를 Quiz 객체로 복원

---

### 5-2. `ScoreManager`
점수 관리 전용 클래스입니다.

#### 주요 속성
- `best_score`: 최고 점수
- `history`: 플레이 기록

#### 주요 메서드
- `update_score(score, total)` : 점수 기록 추가 및 최고 점수 갱신
- `to_dict()` : JSON 저장용 딕셔너리 변환
- `from_dict()` : 저장된 점수 정보 복원

---

### 5-3. `QuizGame`
게임 전체를 관리하는 핵심 클래스입니다.

#### 주요 속성
- `state_file`
- `game_title`
- `quizzes`
- `score_manager`

#### 주요 메서드
- `get_default_quizzes()`
- `build_default_state()`
- `save_state()`
- `load_state()`
- `get_int_input()`
- `get_non_empty_input()`
- `show_menu()`
- `play_quiz()`
- `add_quiz()`
- `list_quizzes()`
- `show_score()`
- `run()`

## 8. 데이터 파일 설명

이 프로젝트는 프로그램 상태를 `state.json` 파일에 저장합니다.

### 저장 위치

* 프로젝트 루트 폴더

### 역할

* 퀴즈 목록 저장
* 최고 점수 저장
* 플레이 기록 저장
* 프로그램 재실행 시 이전 상태 복원

### 저장 형식

UTF-8 인코딩 JSON 형식으로 저장됩니다.

### 예시 스키마

```json
{
    "game": {
        "title": "나라와 수도 맞추기 퀴즈",
        "version": 1
    },
    "score": {
        "best_score": 3,
        "history": [
            {
                "score": 3,
                "total": 6
            }
        ]
    },
    "quiz_data": [
        {
            "question": "대한민국의 수도는 어디일까요?",
            "choices": ["서울", "부산", "인천", "대전"],
            "answer": 1,
            "hint": "한강이 흐르는 도시예요."
        }
    ]
}
```

### 복구 동작

* `state.json` 파일이 없으면 기본 퀴즈 데이터로 새로 생성합니다.
* 파일이 손상되었으면 안내 메시지를 출력한 뒤 기본 데이터로 복구합니다.
